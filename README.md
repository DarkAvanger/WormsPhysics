# WormsPhysics

## Controls

TAB -> Zoom In/Out

W -> Move // Move where you are aiming

A -> Aim Left

D -> Aim Right

Space -> Charge/Fire

R -> Restart when game is done

F1 -> Show map collider

F2 -> Players +1 up to 4, each T pressed adds one player, if pressed 5 times resets players to one

F3 -> End game instantly
